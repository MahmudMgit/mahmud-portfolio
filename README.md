# Mahmud Miam - Portfolio

A sleek and modern one-page developer portfolio built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.  
It showcases my projects, skills, and background as a Computer Science student at De Montfort University.

## Tech Stack
- **Frontend:** React + Vite + Tailwind CSS  
- **Animations:** Framer Motion  
- **Icons:** Lucide React  
- **Deployment:** Vercel  

## Features
- Animated sections with smooth transitions  
- Responsive layout for all devices  
- Modern dark theme with gradient accents  
- Modular skill and project components  

## Project Structure
src/
├── components/ # UI components
├── assets/ # Images & icons
├── App.jsx # Main page
└── index.css # Global styles
